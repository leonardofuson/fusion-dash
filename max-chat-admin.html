Ações rápidas

Ações que você pode disparar daqui sem mexer em código:
  • Disparar cron agora: roda o cron de aplicação de sugestões aprovadas imediatamente, sem esperar segunda 10h BRT.
  • Repopular snapshots: rerun do cron diário on-demand. Útil se algum SQL mudou ou se snapshot ficou strange.
  • Recarregar config: força o backend a re-ler categorias.yaml sem restart (após mexer no GitHub).
  • Nova sugestão: registra uma idéia de melhoria. Sugestões tipo snapshot com SQL pronto, depois de você aprovar, são aplicadas automaticamente pelo cron (segunda 10h BRT, ou clique em "Disparar cron agora"). Tipos palavra_chave / few_shot ficam pendentes pra implementação manual via git push.
